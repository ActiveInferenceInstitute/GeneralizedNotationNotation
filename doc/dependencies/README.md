## Dependency Categories

The GNN project manages dependencies across several categories:

- **Core**: Essential for all functionality
- **Active Inference**: Framework implementations
- **Visualization**: Graphics and diagramming
- **Development**: Testing, linting, documentation
- **Optional**: Advanced features and GPU acceleration
