## X402 Overview

X402 provides:

### Internet-Native Payments
- **HTTP 402 Status Code**: Leverages original "Payment Required" status code
- **Stablecoin Payments**: Direct stablecoin payments over HTTP
- **Instant Settlement**: Fast, frictionless payment processing
- **Built-in Authentication**: Automatic authentication and settlement

### Key Features
- **Programmable Payments**: Payments as easy as data exchange
- **Automatic Settlement**: Seamless integration with web infrastructure
- **Economic Autonomy**: Enable autonomous economic operations for AI agents
- **Global Accessibility**: Internet-native payment rails

## Integration with GNN

X402 integration enables:

- **Agent Economics**: GNN-specified agents can transact autonomously
- **Service Payments**: Pay for APIs, services, and software directly
- **Resource Management**: Autonomous resource consumption management
- **Economic Operations**: Enable economic autonomy for intelligent systems
