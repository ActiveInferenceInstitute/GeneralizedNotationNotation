## Kit Overview

Kit provides:

### Codebase Intelligence
- **Repository Representation**: Navigable codebase representation
- **Symbol Extraction**: Extract symbols and structural elements
- **Advanced Code Search**: Semantic and syntactic code search
- **LLM Integration**: Optimal code context preparation for LLMs

### Key Features
- **MCP Server**: Model Context Protocol server for AI agent interaction
- **Codebase Mapping**: High-level abstractions for code repository interaction
- **Dependency Analysis**: Inter-module dependency analysis
- **Natural Language Summaries**: Code segment summarization

## Integration with GNN

Kit integration enables:

- **GNN Codebase Analysis**: Treat GNN files as specialized source code
- **Symbol Extraction**: Extract GNN structural elements (variables, connections, etc.)
- **Code Search**: Advanced search across GNN models and generated code
- **LLM Enhancement**: Optimal context preparation for LLM-powered GNN tools

## Integration with Pipeline

This documentation is integrated with the 24-step GNN processing pipeline:

1. **Core Processing** (Steps 0-9): GNN parsing, validation, export
   - Kit analysis of GNN codebase
   - Symbol extraction and code search

2. **Simulation** (Steps 10-16): Model execution and analysis
   - Kit analysis of generated code
   - Code intelligence for execution

3. **Integration** (Steps 17-23): System coordination and output
   - Kit results integrated into comprehensive outputs
   - Codebase intelligence reporting

See [src/AGENTS.md](../../src/AGENTS.md) for complete pipeline documentation.
