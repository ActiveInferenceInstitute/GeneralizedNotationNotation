1. **Generative Model**: The model is a generative model that can generate predictions based on input data. It aims to learn from patterns in the data and make predictions with probability.

2. **GNN Representation**: This represents the model as an action-based inference system, where actions are represented by states (represented as vectors) and transitions between states are represented by probabilities of transitioning from one state to another.

3. **Active Inference**: The model is based on active inference theory, which focuses on generating predictions with probability. It aims to learn from patterns in the data using a generative model.

4. **GNN Versions**: There are three versions of this model:
   - Version 1 (Generative Model): Generates predictions based on input data and uses probabilistic graphical models for inference, which is similar to Bayesian inference but with more flexibility.
   - Version 2 (Dynamic Perception Model): Generates predictions using a dynamic perception model that learns from the state-of-the-art in active inference. It also has some limitations compared to version 1.
   - Version 3 (Active Inference): Generates predictions based on input data and uses probabilistic graphical models for inference, which is similar to Bayesian inference but with more flexibility.

5. **Model Parameters**: The model parameters are represented as matrices in the form of a matrix-vector product (MVBP) representation. This allows for easy computation of probabilities and updating of the model based on new data.

6. **Initialization**: The initial parameterizations represent the state space, observation space, and inference space. These can be used to evaluate performance metrics such as accuracy, precision, recall, F1-score, etc.
