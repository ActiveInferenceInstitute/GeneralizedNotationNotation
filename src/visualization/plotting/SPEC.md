# Plotting Utilities — Technical Specification

**Version**: 1.6.0

## Style Configuration

- Font: sans-serif, 10pt
- Figure size: 8×6 inches (default)
- DPI: 150 (publication quality)
- Background: white

## Color Palette

Based on matplotlib's `tab10` with custom harmonization for up to 20 series.

## Save Formats

- PNG (always)
- SVG (optional, `--svg` flag)
- Tight bounding box by default
