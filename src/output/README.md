# Output

## Overview

Test fixture storage for pipeline output artifacts. This is **not** the primary pipeline output target — that is `output/` at the repository root.

## Contents

| Directory | Source Step | Purpose |
|-----------|-----------|---------|
| `2_tests_output/` | Step 2 (Tests) | Test execution fixture data |
| `12_execute_output/` | Step 12 (Execute) | Execution result fixture data |
| `16_analysis_output/` | Step 16 (Analysis) | Analysis result fixture data |

## Usage

This directory is used by the test suite for integration test validation. Contents may be regenerated by test runs.

```bash
# Integration tests that reference these fixtures
uv run pytest src/tests/ -k "output" -v
```

## Important

- Primary pipeline output: `output/` (repository root)
- Fixture copies: `src/output/` (this directory)
- Both follow the same directory structure (`N_step_output/`)
