# RxInfer.jl Execution Module

This submodule handles execution of RxInfer.jl simulations generated by the render stage.

## Overview

This module executes RxInfer.jl simulation scripts, manages Julia environment setup, captures results, and provides comprehensive reporting on reactive inference execution.

## Module Structure

```
src/execute/rxinfer/
├── __init__.py                 # Module initialization
├── README.md                   # This documentation
├── AGENTS.md                   # Detailed agent scaffolding
├── rxinfer_runner.py          # Main execution runner
├── rxinfer_runner.jl          # Julia runner script
└── test_*.py                  # Unit tests
```

## Core Components

### RxInfer Runner (`rxinfer_runner.py`)

**Purpose**: Execute RxInfer.jl simulation scripts

**Key Functions**:
- `run_rxinfer_scripts()` - Execute all RxInfer scripts
- `execute_single_simulation()` - Run single script
- `setup_julia_environment()` - Configure Julia
- `capture_results()` - Collect execution output

### Julia Runner (`rxinfer_runner.jl`)

**Purpose**: Julia-side execution logic

**Features**:
- RxInfer module loading
- Reactive constraint setup
- Inference execution
- Result collection

## Features

## Usage

```python
from execute.rxinfer import run_rxinfer_scripts

results = run_rxinfer_scripts(
    scripts_dir=Path("output/11_render_output"),
    output_dir=Path("output/12_execute_output")
)
```

## Output

Execution produces:
- `execution_results.json` - Execution summary
- `rxinfer_results.json` - Detailed results
- `execution_logs/` - Per-script logs
- `simulation_data/` - Raw simulation outputs
