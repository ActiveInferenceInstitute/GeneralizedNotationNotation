# DisCoPy Executor

Execution module for DisCoPy categorical diagrams generated by the Render step.

## Usage

```python
from execute.discopy import execute_discopy

success = execute_discopy(
    script_path="output/11_render_output/discopy_diagram.py",
    output_dir="output/12_execute_output"
)
```

## Dependencies

- `discopy` (optional, graceful degradation)
